## 프로젝트 구조

`책 이름 / 장. 제목 / 이름`의 양식을 따릅니다.

예) 오브젝트/01장. 객체, 설계/jiae.md

## 커밋 규칙

`[책 이름] 장. 제목 - 이름`의 양식을 따릅니다.

예) [오브젝트] 1장. 객체, 설계 - jiae
